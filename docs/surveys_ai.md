# Apreciaciones IA en Surveys

- Configurar `token_ai` en Configuración del Sistema → Tokens → Inteligencia Artificial.
- La columna IA muestra un ícono cuando existe una apreciación generada.
- Acción “Apreciación IA”:
  - Genera si no existe o si la encuesta se actualizó después de la última apreciación.
  - Muestra estado de carga y notifica éxito o error.
  - Si no hay token configurado, notifica “Token IA no configurado”.
- Ver apreciación: usar la acción “Apreciación IA” o el modal asociado para leer el contenido generado.
