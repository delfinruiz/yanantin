# Filament 4 + Shield + Laravel-Lang

Esta skill contiene directrices para integrar Filament 4 con el plugin Shield para gestión de roles y permisos, así como la configuración de traducciones con Laravel-Lang.

## Descripción General
- **Filament Version**: 4 (Referencia a docs 4.x según usuario: https://filamentphp.com/docs/4.x/introduction/overview)
- **Plugin de Seguridad**: Filament Shield (Roles y Permisos)
- **Traducciones**: Laravel-Lang (Soporte para 2 idiomas)

## Instrucciones de Uso

### 1. Filament Shield
Utilizar para manejar la seguridad de acceso, roles y permisos dentro del panel administrativo.

### 2. Laravel-Lang
Configurado para soportar traducción en dos idiomas. Asegurar que los archivos de idioma estén publicados y configurados en `config/app.php`.

### Referencias
- Documentación provista: https://filamentphp.com/docs/4.x/introduction/overview
